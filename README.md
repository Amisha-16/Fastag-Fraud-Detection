#  FASTag Fraud Detection System

##  Overview

This project focuses on detecting fraudulent FASTag transactions using Machine Learning models. The system analyzes transaction patterns and identifies suspicious activities based on historical data.

---

##  Work Done

In this project, the following steps were performed:

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Selection
* Model Training and Evaluation using multiple algorithms

---

##  Exploratory Data Analysis (EDA)

* Checked missing values and handled them
* Visualized transaction patterns
* Identified fraud vs non-fraud distribution
* Analyzed important features affecting fraud detection

---

##  Data Preprocessing

* Removed/handled null values
* Encoding of categorical variables
* Feature scaling (if applied)
* Outlier handling

---

##  Models Used

The following Machine Learning models were implemented and compared:

* Logistic Regression
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* Decision Tree
* Random Forest

---

##  Model Evaluation

Models were evaluated using:

* Accuracy

The best-performing model was selected based on these metrics.


---

##  How to Run

1. Open Jupyter Notebook or Google Colab
2. Run all cells step-by-step
3. Train and evaluate models

##  Key Insights

* Fraud transactions are usually imbalanced compared to normal transactions
* Certain features strongly influence fraud detection
* Model comparison helps in selecting the best algorithm
